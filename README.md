# youtube-project
python project to analyse and visualise youtube trending videos dataset.
Features:
- Loads and processes YouTube trending video data from a CSV file
- Performs basic and intermediate data analysis
- Generates visualisations including pie charts, histograms, line charts, and bar charts
- Exports selected results to CSV and JSON formats

# How to run
1. Ensure Python 3 is installed
2. Install required libraries:
   - matplotlib
3. Run the main script:
   python main.py
